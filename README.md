# 使用方法
+ 首先运行git clone git@github.com:xxx/xxx.git将所有文件下载到本地
+ 然后 npm install。
+ 之后 输入hexo g生成静态网页，然后输入hexo s打开本地服务器，然后浏览器打开http://localhost:4000/，就可以看到我的博客啦，
